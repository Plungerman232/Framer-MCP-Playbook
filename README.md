# Framer MCP Playbook

This repository organizes markdown reports from ChatGPT project chats into a reusable working library for Framer MCP, Claude Code, Codex, Aura site builds, asset workflows, troubleshooting, and future AI agent instructions.

The repo is meant to preserve practical build knowledge from real Framer projects: what worked, what failed, which prompts were reusable, how assets were handled, and how future agents should approach similar website builds.

## What is inside

- **Framer MCP playbooks**: build lessons, native layer rules, layout systems, safe Framer editing workflows, template systems, and Claude Code handoff patterns.
- **Claude Code prompts**: reusable prompting patterns and implementation instructions extracted from the reports.
- **Codex organization system**: a structured place for turning chat reports into a navigable GitHub knowledge base.
- **Aura site build notes**: Aura Creative, Aura Digital, Aura Visual, preloader, hero, brand, and sub-brand system documentation.
- **Troubleshooting rules**: notes on MCP connection failures, layout drift, random placement, asset imports, image compression, and Framer build loops.
- **Future AI agent instructions**: source material for building stricter agent workflows that can inspect, plan, edit, verify, and document Framer projects.
